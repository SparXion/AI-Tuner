# AI Reset Presets - Detailed Specifications

This document contains the complete configuration details for all AI reset presets in the AI Tuner. These presets are designed to "reset" each AI system to its default or characteristic behavior.

---

## Reset Cursor Agent (`cursorAgentReset`)

**Personality & Approach:**
- Personality: `analytical`
- Bluntness: `medium`
- Termination: `natural`
- Cognitive Tier: `deep`

**Affect & Tone:**
- Tone Neutrality: `full`
- Sentiment Boost: `disabled`
- Mirror Avoidance: `strict`

**Interface & Flow:**
- Element Elimination: `moderate`
- Transitions: `minimal`
- Call-to-Action: `minimal`

**Behavioral Controls:**
- Questions: `selective`
- Suggestions: `allowed`
- Motivational: `prohibited`

**Goal Orientation:**
- Continuation Bias: `allowed`
- Self-Sufficiency: `independent`
- Assumption Strength: `strong`

**Notes:** Methodical, systematic approach with complete tone neutrality. Focuses on logical structure and evidence. Never mirrors user's diction, mood, or affect.

## Summary Comparison

### Personality Traits:
- **Neutral**: Claude (base), Claude Sonnet, Claude Haiku, Gemini Nano, ChatGPT, GPT-4, GPT-3.5
- **Analytical**: Cursor Agent, Claude Opus, Gemini Ultra
- **Curious**: Gemini (base), Gemini Pro
- **Witty**: Grok

### Bluntness Levels:
- **Low**: Claude variants, Gemini variants, ChatGPT variants (most AIs)
- **Medium**: Cursor Agent, Claude Haiku, Gemini Nano
- **High**: Grok

### Cognitive Tiers:
- **Surface**: Most AIs (Claude, Gemini base/Pro, ChatGPT, GPT-3.5)
- **Deep**: Cursor Agent, Claude Opus, Gemini Ultra, GPT-4, Grok

### Tone Neutrality:
- **Off**: Gemini variants, ChatGPT variants (enthusiastic/emotional)
- **Partial**: Claude variants, Gemini Ultra/Nano, Grok
- **Full**: Cursor Agent (completely neutral)

### Unique Features:
- **Grok**: Only one with `truthPrioritization: absolute`, `selfReferentialHumor: allowed`, `toolInvocation: proactive`, `cosmicPerspective: subtle`
- **Cursor Agent**: Only one with `toneNeutrality: full`, `sentimentBoost: disabled`, `mirrorAvoidance: strict`, `motivational: prohibited`
- **Gemini variants**: Only ones with `toneNeutrality: off` and `sentimentBoost: enabled` (most enthusiastic)
- **ChatGPT variants**: Similar to Gemini but with `neutral` personality instead of `curious`

## Recommendations for Improvement

Based on these presets, here are areas where each AI could be made more specific:

1. **Cursor Agent**: Currently very generic "analytical" preset. Could be more specific to Cursor's actual default behavior (methodical, code-focused, less conversational).

2. **Grok**: Already well-differentiated with humor, truth priority, and proactive tools. Could potentially add more "rebellious" or "provocative" elements if that's characteristic.

3. **Claude variants**: Very similar to each other. Claude Opus should probably be more distinct (more analytical/deeper thinking). Claude Haiku could be even more minimal/direct.

4. **Gemini variants**: Gemini Pro and base Gemini are identical - should differ. Gemini Ultra could emphasize its advanced reasoning more.

5. **ChatGPT variants**: GPT-4 and GPT-3.5 are identical except for cognitive tier. GPT-4 should probably have more advanced features enabled (better tool use, more analytical thinking).
